# Node JWT tokens User register/login app

<ul> Technologies Used
  <li>Node</li>
  <li>Express</li>
  <li>JWT Token</li>
  <li>Nodemon</li>
  <li>React Router</li>

## Usage

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5000)

```
npm run server
```

### To build for production

```
npm run build
```
